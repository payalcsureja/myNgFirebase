Direct Source (www.jobcentraldirect.com)

AngularJs Application

Direct Source is platform for career centers to explore jobs and filter them based on specific criteria to help veteran job seekers find jobs. As a development team member, I worked to develop this product from scratch with BA and QA teams.I continue to work on it to make the product more useful by adding new features regularly.

Direct Outreach

Ionic Hybrid Mobile Application

A hybrid mobile app for Direct Source (www.jobcentraldirect.com). Built with Ionic Framework, AngularJS and PHP.

Virtual Career Network (www.vcn.org)

Drupal PHP Application

VCN is drupal based web portal used for career exploration and finding training tools for job seekers in healthcare, green energy and transportation industries.

Portfolio

Angular2 Application

Portfolio is Angular2 application with node/express rest and mongodb/mLab as noSql db. Angular2 application is deployed on firebase and rest api is deployed on heroku with mLab plugin.

Portfolio Monitor

AWS Lamda Canary Function

AWS Lamda canary function to monitor site every five minute and notify admin email verified via AWS SES - Simple Email Service.
